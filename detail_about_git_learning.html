git user.name "username" - when you are new use this to setup your github in local server and put the username as it is in github
git user.email "email" - when you are new use this to setup your github in local server and put the email as it is in github

git clone link_of_file_in_github- to get the repo in your local file from remote file

git status - to check the status of the git file
status have 4 type :- changed staged unchanged

ls -a : to know if it is a github file or not

git push main origin - to push the changed local file to the remote repo

git init - to initialize the non github file as github file but in local
