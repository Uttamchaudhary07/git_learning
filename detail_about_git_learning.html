git user.name "username" - when you are new use this to setup your github in local server and put the username as it is in github
git user.email "email" - when you are new use this to setup your github in local server and put the email as it is in github

git clone link_of_file_in_github- to get the repo in your local file from remote file

git status - to check the status of the git file
status have 4 type :- changed staged unchanged

git commit -m "write the readme about the file here" - write the description aboout file and run.

ls -a : to know if it is a github file or not

git push origin main - to push the changed local file to the remote repo (you can change the origin name as well)

git init - to initialize the non github file as github file but in local

git remote add orgin link : you can change origin to anything and add link in place of link

git branch - to know in which branch you are at the moment

git branch -M main - to rename branch (main can be changed to any name like new so the branch will be renamed to new)

git chackout branchname - to navigate or to go to the required branchname

git checkout -b new branch name - to create new branch

git branch -d branch name- delete branch
